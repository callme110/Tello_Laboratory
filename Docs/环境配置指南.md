## 3. 后端环境配置（Python）

### 3.1 安装 Python

从 [python.org](https://www.python.org/downloads/) 下载并安装 Python 3.8+。

安装时勾选 **"Add Python to PATH"**。

验证安装：

```bash
python --version
```

### 3.2 创建虚拟环境（推荐）

```bash
# 在项目根目录下执行
python -m venv .venv

# 激活虚拟环境
# Windows:
.venv\Scripts\activate

# macOS / Linux:
source .venv/bin/activate
```

### 3.3 安装 Python 依赖

```bash
pip install djitellopy opencv-python numpy flask flask-cors flask-socketio
```

各依赖说明：

| 包名 | 版本建议 | 用途 |
|------|---------|------|
| `djitellopy` | >= 2.4.0 | Tello 无人机 SDK，封装 UDP/TCP 通信 |
| `opencv-python` | >= 4.5.0 | 图像处理（颜色识别、轮廓检测等） |
| `numpy` | >= 1.21.0 | 数值计算，用于 PD 控制和图像矩阵运算 |
| `flask` | >= 2.0 | Web 服务器，提供 REST API |
| `flask-cors` | >= 3.0 | 跨域请求支持（前端访问后端必需） |
| `flask-socketio` | >= 5.0 | WebSocket 支持，用于实时状态推送 |

完整安装命令（指定最低版本）：

```bash
pip install "djitellopy>=2.4.0" "opencv-python>=4.5.0" "numpy>=1.21.0" \
            "flask>=2.0" "flask-cors>=3.0" "flask-socketio>=5.0"
```

### 3.4 Haar 级联分类器文件（人脸追踪功能）

人脸追踪功能依赖 `haarcascade_frontalface_default.xml`，已存放于：

```
Resource/haarcascade_frontalface_default.xml
```

如文件缺失，可从 OpenCV 官方仓库下载：
```
https://github.com/opencv/opencv/tree/master/data/haarcascades
```

> **注意：** 使用 `FaceTracking.py` 前，请根据实际路径修改脚本中的 `cascade_path` 变量。

### 3.5 Jupyter Notebook（可选，用于单功能测试）

```bash
pip install jupyter
```

## 4. 前端环境配置（Node.js）

前端使用 **React 18 + TypeScript + Vite 6 + TailwindCSS 4** 技术栈，包管理器为 **pnpm**。

### 4.1 安装 Node.js

从 [nodejs.org](https://nodejs.org/) 下载并安装 Node.js 20.x LTS。

验证安装：

```bash
node --version   # v20.x.x
npm --version
```

### 4.2 安装 pnpm

```bash
npm install -g pnpm
pnpm --version   # 8.x 或以上
```

### 4.3 安装前端依赖

```bash
cd Frontend
pnpm install
```

主要依赖说明：

| 包名 | 版本 | 用途 |
|------|------|------|
| `react` / `react-dom` | 18.3.1 | UI 框架 |
| `vite` | 6.3.5 | 构建工具 / 开发服务器 |
| `tailwindcss` | 4.1.12 | 原子化 CSS |
| `socket.io-client` | 4.x | WebSocket 客户端，接收实时状态 |
| `@radix-ui/*` | 各版本 | shadcn/ui 底层无障碍组件库 |
| `lucide-react` | 0.487.0 | 图标库 |

## 6. 运行说明

### 6.1 启动前端控制界面（主要使用方式）

控制界面需要前后端同时运行：

```bash
# 终端 1：启动后端 API + WebSocket 服务
cd Frontend
python backend_example.py dev

# 终端 2：启动前端开发服务器
cd Frontend
pnpm dev
```

浏览器访问：`http://localhost:5173`

> **`dev` 参数说明：** 加 `dev` 参数为开发模式，仅启动 Flask API（端口 5000），不启动 pywebview 窗口，前端独立运行在 5173 端口。

### 6.2 单独运行后端 Python 脚本

不依赖前端，直接在终端运行：

```bash
# 示例：运行键盘控制
cd Backend/control
python KeyboardControl.py

# 示例：运行人脸追踪
cd Backend/vision
python FaceTracking.py

# 示例：运行基础动作
cd Backend/basic
python BasicMovements.py
```

### 6.3 颜色参数标定（穿环 / 巡线前必做）

穿环和巡线功能依赖 HSV 颜色阈值，使用前需针对实际场景标定：

```bash
cd Backend/vision
python ColorPicker.py
```

将标定结果填入对应参数：
- 穿环：`Backend/projects/ring_task.py` → `HSV_LOWER` / `HSV_UPPER`
- 巡线：`Backend/projects/LineFollower.py` → `HSV_LOWER` / `HSV_UPPER`

## 7. 常见问题

**Q: 连接无人机后脚本报错 `Connection refused`？**

A: 确认已连接 Tello Wi-Fi，且无其他程序占用 UDP 8889 端口。Tello 默认 IP 为 `192.168.10.1`。

---

**Q: `import cv2` 报错？**

A: 重新安装 OpenCV：
```bash
pip uninstall opencv-python
pip install opencv-python
```

---

**Q: `pnpm install` 报网络错误？**

A: 切换国内镜像源：
```bash
pnpm config set registry https://registry.npmmirror.com
pnpm install
```

---

**Q: 前端页面空白或报 `Cannot find module`？**

A: 确认已执行 `pnpm install`，且 Node.js 版本 >= 18：
```bash
node --version
cd Frontend && pnpm install && pnpm dev
```

---

**Q: 穿环 / 巡线启动后无人机没有反应？**

A: 检查以下几点：
1. 确认无人机已起飞（穿环 / 巡线任务不会自动起飞）
2. 确认已调用 `streamon`（后端 `line/start` 端点会自动调用）
3. 用 `ColorPicker.py` 重新标定颜色参数

---

**Q: 巡线时无人机抖动严重？**

A: 调整 `LineFollower.py` 中的 PD 参数：
- 抖动 / 蛇形 → 降低 `KP_YAW`（如从 0.20 降到 0.12）
- 反应迟钝 → 提高 `KP_YAW`；转弯超调 → 提高 `KD_YAW`

---

**Q: 前端 WebSocket 无法接收状态推送？**

A: 确认后端使用 `flask-socketio` 启动（`python backend_example.py dev`），不要用普通 `flask run`。检查浏览器控制台是否有跨域错误。
